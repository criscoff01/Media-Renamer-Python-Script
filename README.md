#📺 Episode Renamer (Python)

A lightweight, configurable Python script for batch renaming TV episodes in a directory. Designed as a flexible alternative to tools like Microsoft PowerRename, with a focus on automation and repeatability.

🚀 Features
Batch renaming for episode files
Config-driven (easy to reuse per show)
Natural file sorting (fixes 1, 2, 10 ordering issues)
Custom naming patterns (e.g., S01E01)
Safe dry-run mode for previewing changes
Supports common video file formats (.mp4, .mkv, .avi)
